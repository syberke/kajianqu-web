# Role dan Permission

| Aksi | Siswa | Asatidz approved | Admin |
|---|---:|---:|---:|
| Membaca materi published | Ya | Ya | Ya |
| Latihan AI Quran | Ya | Ya | Opsional |
| Bergabung kelas | Ya | Tidak | Kelola |
| Membuat materi | Tidak | Ya | Tidak |
| Publish materi sendiri | Tidak | Tidak | Ya |
| Mengelola kelas | Tidak | Ya | Ya |
| Melihat fee sendiri | Tidak | Ya | Ya |
| Review/verifikasi | Tidak | Tidak | Ya |
| Pengaturan sistem | Tidak | Tidak | Ya |

Permission di frontend hanya untuk pengalaman pengguna. RLS dan fungsi server tetap menjadi sumber otorisasi.

Asatidz non-approved hanya boleh melengkapi profil, dokumen, dan melihat status verifikasi.
